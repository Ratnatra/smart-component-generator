# GitHub Pages Deployment

This project is a static website and is ready to be hosted via GitHub Pages.

## Deployment steps
1. Push all changes to the main branch.
2. Open the GitHub repository.
3. Go to Settings > Pages.
4. Under Source, choose Deploy from a branch.
5. Select main as the branch and /(root) as the folder.
6. Save and wait for the deployment to complete.
